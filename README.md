# ESP-32 Weather Station

## Components
● ESP-WROOM-32 Dev Board
● Breadboard
● DHT-22 sensor
● 0.96’’ 128x64 pixels Yellow-Blue I2C OLED Display Module
● Micro USB to USB Type A data cable
